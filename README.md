# UCA Emprende · Preview de diseño

Sitio estático interno con las maquetas de diseño de **UCA Emprende** elaboradas por **mowomo**.

> Repositorio privado. Acceso restringido al equipo y al cliente con el link de GitHub Pages.

## Contenido

- `index.html` — landing con los enlaces a las maquetas
- `home-v2.html` — Home, versión v2
- `actividades.html` — Listado de actividades, v1

## Servir local

```bash
python3 -m http.server 8000
# abrir http://localhost:8000
```
